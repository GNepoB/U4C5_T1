## Funcionamento
- O sistema inicializa com o LED vermelho aceso.
- A cada 3 segundos, o semáforo muda de estado na seguinte ordem:
  Vermelho -> Amarelo
  Amarelo -> Verde
  Verde -> Vermelho
- No terminal, mensagens são exibidas a cada segundo para indicar o estado atual do semáforo.
